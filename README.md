Call Center Performance Dashboard

2. Project Overview
Built an end-to-end interactive dashboard in Microsoft Excel analyzing over 32,000 real-world call center records. The project uncovers key operational insights including SLA response performance, customer sentiment trends, call channel distribution, and regional volume breakdown across 4 call centers. The dashboard features dynamic slicers enabling drill-down filtering by call reason, location, and response type — transforming raw data into actionable business intelligence.

3. Problem Statement
How can a call center identify bottlenecks in response time, understand customer sentiment, and improve service delivery across multiple locations? This project aims to answer that by analyzing over 32,941 customer interaction records and visualizing the findings in an interactive dashboard.

4. Dataset

Source: Real World Fake Data
Records: 32,941
Key Columns: Customer ID, Sentiment, CSAT Score, Call Timestamp, Call Day, Reason, City, State, Channel, Response Time, Call Duration, Call Center


5. Key Findings

🔴 71% of all calls were related to Billing Questions — the single largest driver of contact volume
😞 52%+ of customer interactions recorded Negative or Very Negative sentiment
⏱️ 62% of calls were handled Within SLA, but 25% fell Below SLA — highlighting a response efficiency gap
📍 Los Angeles/CA handled the highest call volume at 13,734 calls
🕐 Average call duration was 25 minutes, suggesting complex interactions that may benefit from better self-service resources


6. Dashboard Features

📈 Call trend line chart (day-by-day volume tracking)
⏱️ SLA response horizontal bar chart (Above / Below / Within)
😊 Customer sentiment pie chart
📞 Call channel donut chart (Call-Center, Chatbot, Email, Web)
🗺️ Geographic map view by state
🎛️ Dynamic slicers — filter by Reason, Call Center location, and Response type


7. Tools Used

Microsoft Excel — Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, Map Chart

8. How to Use

Download the .xlsx file from this repository
Open it in Microsoft Excel (2016 or later recommended)
Navigate to the Dashboard tab
Use the slicers on the right panel to filter by Reason, Call Center, or Response type
All charts update automatically based on your selection
